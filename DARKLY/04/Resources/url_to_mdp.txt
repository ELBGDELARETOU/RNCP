http://localhost:8080/?page=../../../../../../../../../etc/passwd

explication : 

Ce qui se passe ici c est un LFI (local file inclusion), c est un test classique pour tester la vulnerabilite et la bonne gestion
su systeme de page sur le site. La methode $_GET qui permet de naviguer entre les pages n est pas proteger, on peut donc facilement 
remonter dans l arboresence du serveur pour avoir acces au fichier du site ex: etc/pass, var/log etc ...
Pourquoi /etc/passwd ? Car c est un fichier present sur toute les serveurs Linux.
